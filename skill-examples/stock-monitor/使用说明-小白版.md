# 股票监控助手 - 使用说明

## 🚀 3步上手

### 第1步：下载解压
下载后解压到桌面，你会看到一个文件夹。

### 第2步：配置股票
打开文件夹里的 `config` 文件夹，找到 `stocks.json.example`：

**Windows:**
1. 右键点击 `stocks.json.example`
2. 复制 → 粘贴（会在同目录生成副本）
3. 把副本重命名为 `stocks.json`
4. 右键 `stocks.json` → 打开方式 → 记事本

**Mac:**
1. 复制 `stocks.json.example`
2. 粘贴，重命名为 `stocks.json`
3. 双击用文本编辑打开

**修改内容（照葫芦画瓢）：**
```
把这一行："code": "600519",
改成你的："code": "你的股票代码",

把这一行："cost": 1500.00,
改成你的："cost": 你的成本价,
```

**例子：** 如果你买了茅台，成本1480元
```json
{
  "code": "600519",
  "name": "茅台",
  "cost": 1480.00,
  "shares": 100
}
```

改完保存关闭。

### 第3步：启动监控

**Windows:** 双击 `启动监控.bat`
**Mac:** 双击 `启动监控.command`

看到黑色窗口显示「监控已启动」就行了！

**重要：** 不要关闭这个黑窗口，最小化它，让它在后台运行。

## 📱 设置飞书通知（可选）

不想一直看黑窗口？让程序发消息到你手机：

### 1. 创建飞书机器人
1. 打开飞书，进入一个群（或新建一个自己的群）
2. 点击右上角「设置」⚙️
3. 找到「群机器人」→「添加机器人」
4. 选「自定义机器人」
5. 起个名字，比如「股票助手」
6. 复制Webhook地址（很长一串链接）

### 2. 配置到程序
打开 `config/stocks.json`，找到 `"notifyChannels": ["console"]`

改成：
```json
"notifyChannels": ["feishu"],
"feishu": {
  "webhook": "你刚才复制的链接"
}
```

保存，重启程序。

### 3. 测试
把止损线设低一点（比如1%），等股价波动触发预警，看手机能不能收到飞书消息。

## ❓ 常见问题

**Q: 双击没反应？**
A: 右键 `启动监控.bat` → 以管理员身份运行

**Q: 提示"不是内部或外部命令"？**
A: 第一次运行需要安装环境，多等几分钟，不要关窗口

**Q: 怎么停止监控？**
A: 关掉黑窗口就行

**Q: 怎么改股票？**
A: 关掉程序 → 改 `stocks.json` → 重新启动

**Q: 能监控几只股票？**
A: 想加几只加几只，格式一样复制粘贴就行

**Q: 电脑关了还监控吗？**
A: 不行，需要电脑一直开着
