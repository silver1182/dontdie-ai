# 飞书机器人配置教程（图文版）

## 📱 为什么要配置飞书通知？

配置后，当股价触发预警时，你的飞书会收到消息提醒，不用一直盯着电脑。

**效果：**
```
🚨 止损预警 - 茅台
茅台(600519) 跌幅5.2%
当前: ¥1425 | 成本: ¥1500
盈亏: -¥7500 (-5.0%)
止损线: -5%
```

---

## 🔧 配置步骤（5步）

### 第1步：进入群设置

打开飞书，进入你想接收通知的群（可以新建一个只有自己的群）。

点击右上角的 **「...」** 图标：

[截图1：点击右上角菜单]

选择 **「设置」**：

[截图2：点击设置]

---

### 第2步：添加机器人

在设置页面，找到 **「群机器人」**，点击进入：

[截图3：点击群机器人]

点击 **「添加机器人」**：

[截图4：点击添加机器人按钮]

---

### 第3步：选择机器人类型

选择 **「自定义机器人」**（图标是橙色的）。

点击 **「添加」**。

---

### 第4步：设置机器人信息

填写机器人信息：

[截图5：填写机器人信息]

| 字段 | 填写内容 | 示例 |
|------|----------|------|
| 机器人名称 | 起一个容易认的名字 | 股票助手 |
| 描述 | 简单说明用途 | 股票监控预警通知 |
| 头像 | 可以选一个喜欢的颜色 | 绿色 |

填完后点击 **「添加」**。

---

### 第5步：复制Webhook地址

添加成功后，会显示一个 **Webhook 地址**：

[截图6：复制Webhook地址]

**⚠️ 重要：**
- 这个地址是唯一的，相当于机器人的「电话号码」
- 点击「复制」按钮复制完整链接
- 不要泄露给别人，否则别人也能发消息到你的群

**地址长这样：**
```
https://open.feishu.cn/open-apis/bot/v2/hook/xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
```

## 💾 填入股票监控程序

### 方式1：在GUI界面中填写

1. 打开股票监控助手网页
2. 找到「飞书通知设置」
3. 粘贴刚才复制的Webhook地址
4. 点击「保存设置」

[界面截图位置]

### 方式2：直接修改配置文件

如果用命令行版，打开 `config/stocks.json`：

```json
{
  "settings": {
    "notifyChannels": ["feishu"],
    "feishu": {
      "webhook": "https://open.feishu.cn/open-apis/bot/v2/hook/你的地址"
    }
  }
}
```

把复制的地址粘贴到 `"webhook":` 后面的引号里。

## 🧪 测试通知

配置完成后，测试一下是否能收到消息：

### 方法1：手动触发测试

在程序界面点击「测试通知」按钮（如果有的话）。

### 方法2：调整止损线触发

1. 把某只股票的止损线设为 **1%**（很低，很容易触发）
2. 等待股价波动
3. 如果跌幅超过1%，应该能收到飞书消息

### 方法3：命令行测试

如果懂一点技术，可以用curl测试：

```bash
curl -X POST -H "Content-Type: application/json" \
  -d '{"msg_type":"text","content":{"text":"测试消息"}}' \
  https://open.feishu.cn/open-apis/bot/v2/hook/你的地址
```

发送后，飞书群应该收到「测试消息」。

## ❓ 常见问题

**Q: 收不到消息？**

检查清单：
- [ ] Webhook地址是否复制完整？（要包含https://开头）
- [ ] 是否在程序中点击了「保存」？
- [ ] 飞书群是否禁言了机器人？
- [ ] 止损/止盈条件是否达到了？

**Q: 可以发到多个群吗？**

可以，需要在每个群都添加机器人，得到不同的Webhook地址，然后在程序里配置多个通知渠道。

**Q: Webhook地址泄露了怎么办？**

在飞书群里找到这个机器人，删除它，重新添加一个新的，Webhook地址会变。

**Q: 手机能收到通知吗？**

可以，飞书APP会推送消息，和收到普通消息一样。
