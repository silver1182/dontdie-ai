# 🌸 明妃三层记忆系统 - 完整版

基于 EverMemOS 架构设计的本地轻量化记忆系统

## 🏗️ 三层架构

```
┌─────────────────────────────────────────────────────────┐
│                    明妃三层记忆系统                        │
├──────────────┬──────────────┬───────────────────────────┤
│   第一层      │   第二层      │        第三层             │
│  原始记忆层   │  向量记忆层   │      画像层               │
├──────────────┼──────────────┼───────────────────────────┤
│ 📝 完整对话   │ 🧠 语义向量   │    👤 用户画像            │
│ SQLite存储   │ Ollama生成   │   自动构建                │
│ 关键词搜索   │ 余弦相似度   │   偏好学习                │
├──────────────┼──────────────┼───────────────────────────┤
│ 54条记忆     │ 18条向量     │   动态生成                │
└──────────────┴──────────────┴───────────────────────────┘
```

## 🚀 快速使用

```python
from memory_system_complete import remember, recall, get_profile

# 存入记忆（同时入三层）
remember("user", "我喜欢看龙族小说", user_id="silver1182")

# 语义召回（向量+关键词混合）
context = recall("龙族是什么", user_id="silver1182")
print(context)
# 输出:
# ## 相关记忆
# 🧠向量记忆 [0.95] [2026-03-04] 我喜欢看龙族小说...

# 查看画像
print(get_profile("silver1182"))
# 输出:
# ## 用户画像 (v1)
# **兴趣**: 龙族, 小说, 编程, AI...
# **特征**: 热爱小说, 技术爱好者
```

## 🔧 技术栈

- **原始层**: Python + SQLite
- **向量层**: Ollama (nomic-embed-text) + 余弦相似度
- **画像层**: 规则提取 + 统计分析

## 📈 Phase 3 画像层功能

- ✅ 兴趣分布分析
